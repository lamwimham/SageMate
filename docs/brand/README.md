# SageMate Logo

## Direction: Minimal Second Brain

SageMate 的主 logo 采用 **outline 人脑** 方向，表达产品愿景：一个本地优先、持续生长、服务个人 Agent 的第二大脑。

## Concept

- **Human brain outline**: 直观表达“第二大脑”，比抽象 AI 符号更贴近产品核心。
- **S-shaped fold**: 中央脑回隐含 SageMate 的 `S`，也代表个人知识索引路径。
- **Minimal strokes**: 去掉多节点、雷达弧、复杂渐变，保证 favicon、桌面图标、插件图标的小尺寸可读性。
- **Outline style**: 更克制、耐看，能融入当前深色、低噪声、知识工作者导向的产品调性。

## Files

- `sagemate-logo-brain-outline.svg`: primary outline mark.
- `sagemate-logo-brain-outline-mono.svg`: one-color version using `currentColor`.
- `sagemate-logo-brain-outline-app-icon.svg`: app icon source on deep background.

- `src-tauri/icons/*`

## Palette

- Deep background: `#0C0C12`
- Neural indigo: `#818CF8`
- Soft indigo: `#C7D2FE`

## Usage Guidance

- Use the **outline mark** on product pages, documentation, and UI surfaces where the background is already controlled.
- Use the **app icon source** for launcher icons, browser extension icons, and favicon generation.
- Use the **mono version** in UI components that need to inherit text color.
