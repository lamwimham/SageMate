## 关键概念对照表

| 概念 | 含义 |
|------|------|
| 第二大脑 | 你的个人知识库 + 索引系统 + Taste 图谱 |
| 索引思维 | AI 时代，建立索引比掌握知识更重要 |
| 灵魂数据 | 主动收藏、反复回顾、编辑痕迹、双向链接、忽略内容 |
| 工作习惯数据 | 日历、邮件、点击行为、效率模式（现有 AI 助手收集的） |
| Taste | 你的品味、偏好、审美、价值观、热情、盲区 |
| 同频 | Agent 的回答风格、关注维度、盲区与你的真实自我一致 |
