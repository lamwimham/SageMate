> 记录时间：2026-04-25
> 来源：用户灵感

---

## 核心洞察（用户原话）

> "一个 agent 的独特性，正是你日积月累的喂他你喜欢的一切，他才会变得和你同频，而这个和现在的多数 agent 不一样，他们只是提取你的工作习惯，他们学习到的，往往是你为了达到某种目的而刻意遵守的规则，而不是获取你的品味，你的兴趣，你的爱好，你的灵魂。"

---

## 洞察拆解

### 第一层：现有 Agent 学什么？—— 工作习惯

**现有 AI 助手（Copilot、Claude Projects、Google Assistant）的学习对象：**

| 数据类型 | 例子 | 反映什么 |
|---------|------|---------|
| 日历数据 | 你几点开会、会议时长 | 时间管理习惯 |
| 邮件往来 | 你回复的速度、措辞风格 | 沟通习惯 |
| 文档编辑 | 你怎么写报告、怎么改稿子 | 工作模式 |
| 点击行为 | 你常用哪些功能、跳过哪些 | 效率偏好 |
| 搜索历史 | 你查什么关键词 | 即时需求 |

**这些数据的本质：**
- 是**行为痕迹**，不是**内心痕迹**
- 是**为了完成任务的刻意表现**，不是**自然状态下的真实自我**
- 是"你怎么工作"，不是"你是谁"

**举例：**
- 你在邮件里用正式语气，不代表你是个正式的人，那只是职场规则
- 你快速回复消息，不代表你喜欢即时沟通，那可能只是不想让对方等
- 你的日历排满了会议，不代表你热爱开会，那可能只是岗位职责

### 第二层：SageMate 第二大脑收集什么？—— 灵魂数据

**SageMate 收集的数据类型：**

| 数据类型 | 例子 | 反映什么 |
|---------|------|---------|
| 主动收藏 | 凌晨三点还在读的小众文章 | 真正的好奇心方向 |
| 反复回顾 | 某篇笔记你一个月看了 8 次 | 深度兴趣点 |
| 编辑痕迹 | 你删改了哪句话、怎么重组段落 | 思考方式和审美 |
| 双向链接 | 你把 A 概念和 B 概念连在了一起 | 独特的认知结构 |
| 忽略的内容 | 你收藏了又删除的、标记为不重要的 | 价值判断和品味 |
| AI 编译后的修改 | 你调整了 AI 提炼的哪个部分 | 你对"什么是重要的"的定义 |
| 提问模式 | 你问 AI 什么问题、怎么问 | 你的求知欲和思维方式 |

**这些数据的本质：**
- 是**主动选择**，不是被动痕迹
- 是**无目的的真实行为**，不是有目的的刻意表现
- 是"你是谁"，不是"你怎么工作"

**关键区别：**

```
工作习惯数据          灵魂数据
─────────────────    ─────────────────
你回复邮件很快    →   你在凌晨三点还在读某篇哲学散文
你日历排很满      →   你反复回看某张知识卡片 8 次
你用正式措辞      →   你收藏了又删掉了一首诗
你常用某个功能    →   你把两个看似无关的概念连在了一起
                     
"你怎么完成任务"     "什么触动了你"
"职场中的你"         "独处时的你"
"刻意表现"           "自然流露"
"行为"               "灵魂"
```

### 第三层：为什么"灵魂数据"才能让 Agent "同频"

**同频的定义：**
不是"它能帮你完成任务"，而是"它理解你为什么想做这件事"。

**举例对比：**

| 场景 | 基于工作习惯的 Agent | 基于灵魂数据的 Agent |
|------|---------------------|---------------------|
| 你搜"微调" | 帮你找最热门的微调教程 | 知道你对 LoRA 比对全参数微调更感兴趣，因为你收藏的 80% 文章都是 LoRA 相关的 |
| 你写一篇文章 | 帮你按标准结构写 | 知道你喜欢先抛观点再论证，因为你所有笔记都是这个结构 |
| 你问"推荐一本书" | 推荐豆瓣评分最高的 | 推荐一本你收藏夹里某篇文章引用过的小众书，因为那篇文章你反复看了 5 次 |
| 你情绪低落 | 建议你休息或运动 | 知道你每次低落都会重读某类散文，主动把那篇推给你 |

**关键洞察：**
> 工作习惯让 Agent **高效**。
> 灵魂数据让 Agent **懂你**。

### 第四层：日积月累的力量

**用户原话中的关键词："日积月累"**

Agent 的独特性不是一次性喂出来的，是长期累积的结果：

| 时间维度 | 灵魂数据的累积效果 |
|---------|------------------|
| Day 1 | Agent 知道你收藏了一篇文章 |
| Day 7 | Agent 发现你收藏了 5 篇同主题文章，开始识别兴趣方向 |
| Day 30 | Agent 发现你反复回顾某些卡片，识别出深度兴趣点 |
| Day 90 | Agent 发现你的链接模式，理解你的认知结构 |
| Day 365 | Agent 知道你的季节性生活节奏、周期性兴趣变化 |
| Day 1000 | Agent 比你更清楚"你是谁"，因为它没有自我欺骗 |

**关键洞察：**
> Agent 的同频程度 = 你喂它灵魂数据的时间 × 数据的纯度。
>
> 工作习惯数据是"速溶咖啡"——快，但没有层次。
> 灵魂数据是"陈年老酒"——慢，但有深度。

| **忽略/删除** | 不是"不感兴趣"，是**价值判断**——"这个不配进入我的大脑" |
| **提问模式** | 不是搜索关键词，是**求知欲的方向** |

### 竞品在做什么？—— 工作习惯

| 产品 | 收集的数据 | 本质 |

| Microsoft Copilot | 邮件、日历、文档、会议记录 | 工作习惯 |

> 我们做"灵魂镜像"。

## 产品化方向

### 方向1：灵魂数据仪表盘（Soul Dashboard）

**功能：**
不是显示"你本周处理了多少篇文章"，而是显示：

```
你的灵魂指纹（基于 365 天的数据）：

深度兴趣点：
  ▓▓▓▓▓▓▓▓ LoRA 微调（你反复回顾 32 次）
  ▓▓▓▓▓▓ 独立开发（你主动收藏 28 篇）
  ▓▓▓▓ 存在主义哲学（你建立了 15 个双向链接）

好奇心方向（近期新增）：
  ▓▓▓ 图神经网络（过去 30 天收藏 12 篇）

价值判断（你忽略什么）：
  ✕ 商业鸡汤类文章（收藏后又删除 89%）
  ✕ 热点追踪类内容（从未收藏）

思考方式：
  你喜欢"先建立框架，再填充细节"（90% 的笔记都是这个结构）
  你偏好"对比分析"（60% 的笔记包含"vs"）
```

**文案钩子：**
> "这不是效率报告。这是你灵魂的 X 光片。"

### 方向2：Agent 同频度评分

**功能：**
显示"你的 Agent 现在有多像你"：

```
Agent 同频度：67%

它懂你的：
  ✓ 技术深度偏好
  ✓ 独立开发的热情
  ✗ 你对某类音乐的情感连接（数据不足）
  ✗ 你的季节性情绪波动（需要更多时间）

建议：多收藏一些你真正热爱的非技术类内容，
让 Agent 了解工作之外的你。
```

### 方向3："工作模式" vs "灵魂模式" 切换

**功能：**
用户可以选择让 Agent 基于哪种数据回答：

- **工作模式**：基于日历、邮件、任务数据（高效完成任务）
- **灵魂模式**：基于收藏、笔记、链接数据（懂你、有温度）

**使用场景：**
- 写工作邮件 → 工作模式
- 写个人博客 → 灵魂模式
- 做决策分析 → 灵魂模式（因为它知道你的真实偏好，不是职场面具）

### 方向4：灵魂数据时间轴

**功能：**
显示用户"灵魂数据"的积累过程：

```
你的 Agent 成长日记：

2025.06：Agent 发现你对 LoRA 感兴趣
2025.08：Agent 发现你反复回顾某篇哲学笔记
2025.10：Agent 发现你把技术和哲学连在了一起
2025.12：Agent 第一次写出了"像你的话"
2026.03：Agent 同频度达到 60%
2026.06：Agent 比你更清楚"你是谁"
```
