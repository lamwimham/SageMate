# SageMate 付费等级权益规格书

> **核心原则**：
> - **买断** = 软件卖给你，功能全开、用量无限，**AI 算力你自己搞定**。
> - **订阅** = 软件 + AI 算力打包服务，**我们负责 LLM/OCR 选型和接口**，你只管用。
> - **Free** = 体验版，功能受限，AI 算力要么自己配，要么用少量赠送额度。

### 1.1 买断制 = 卖软件

用户付 ¥98，软件终身归他。
- ✅ 所有功能可用
- ✅ 无限用量
- ✅ 终身更新
- ❌ 我们不提供 LLM/OCR 服务
- ❌ 用户需要自己买 API key、自己配模型

**类比**：买了 Photoshop，但打印要去图文店自己付费。

### 1.2 订阅制 = 卖软件 + AI 算力托管服务

用户付 ¥15/月，我们提供完整托管。
- ✅ 所有功能可用
- ✅ 有上限但足够日常使用的用量
- ✅ **内置 LLM 接口**，无需配置
- ✅ **内置 OCR 接口**，无需配置
- ✅ 我们负责模型选型、接口稳定性、故障切换
- ❌ 到期不续费，回 Free 模式

**类比**：买了 iPhone + iCloud，硬件是你的，但云服务按月续费。

### 1.3 为什么订阅制必须限制用量？

因为 **订阅费包含了 AI 算力成本**。

| 成本项 | 买断用户 | 订阅用户 |
|--------|---------|---------|
| 软件开发 | 一次性覆盖 | 订阅费覆盖 |
| LLM API | 用户自付 | **我们承担** |
| OCR API | 用户自付 | **我们承担** |
| 模型维护 | 用户自维护 | **我们维护** |

如果我们给订阅用户"无限 LLM 调用"，一个重度用户一个月可能消耗 ¥500+ 的 token，我们只收 ¥15，会亏死。

所以订阅制的用量限制不是"故意恶心用户"，而是**成本边界**。

### 4.4 成本控制策略

| 策略 | 实现 | 效果 |
|------|------|------|
| **模型选型** | 默认 qwen-plus（性价比高） | 控制单位 token 成本 |
| **上下文截断** | 超过 40000 字符自动截断 | 减少无效 token |
| **并发限制** | 订阅用户编译队列串行 | 防止 burst |
| **用量上限** | 硬限制 50 编译/200 查询/天 | 封顶成本 |
| **降级策略** | 用量超限后 fallback 到本地模型（如有） | 保证基本可用 |

**单订阅用户月成本估算**（按上限使用）：
- 50 编译/天 × 30 天 = 1500 编译/月
- 每次编译平均消耗 ¥0.02（qwen-plus）
- 月 LLM 成本 ≈ ¥30
- 订阅费 ¥15/月 → **亏损 ¥15/月**

**现实情况**：大多数用户不会每天用满上限。平均使用率约 30%。
- 实际月 LLM 成本 ≈ ¥9
- 订阅费 ¥15/月 → **毛利 ¥6/月**

**结论**：¥15/月的定价在平均使用场景下可以 cover 成本。重度用户可能会亏，但会被买断版（¥98）转化。

### 5.1 赠送额度

Free 用户首次启动时，**自动赠送**：
- LLM 调用：20 次（编译 + 查询共用）
- OCR 调用：10 次

用完之后：
```
您的免费 LLM 额度已用完（20/20）。
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
选择 1：配置自己的 LLM API key（免费继续用）
选择 2：升级到订阅版（¥15/月，无需配置）
选择 3：升级到买断版（¥98，终身无限）
```

### 5.2 OCR 的分级设计

| Tier | PDF 处理方式 | 覆盖场景 |
|------|-------------|---------|
| Free | **Poppler 纯文本提取** | 只能处理可选中文字的 PDF |
| Free | 扫描版/图片 PDF → 提示"需 Pro" | 引导升级 |
| 买断 | Poppler + 自配 OCR | 扫描版 PDF 可识别 |
| 订阅 | Poppler + 内置 OCR | 扫描版 PDF 可识别 |

**为什么 OCR 要分级？**
- OCR 调用成本高（GLM-OCR 按页计费，一本扫描版书可能 ¥5~20）
- 纯文本 PDF 提取成本低（本地 Poppler，零成本）
- 扫描版 PDF 的识别是"魔法"，值得付费

**用户上传扫描版 PDF 时的提示（Free）**：
```
⚠️ 此 PDF 为扫描版（图片格式），无法提取文字。

解锁 OCR 识别：
• 升级到 Pro（订阅 ¥15/月 或 买断 ¥98）
• 或配置自己的 OCR API key

[升级] [配置 OCR] [取消]
```

### 5.3 为什么 Free 不能一直用内置 LLM？

- 如果 Free 用户无限使用内置 LLM，会被大量薅羊毛
- 但完全不给 LLM 体验，用户无法感受产品价值
- **赠送额度 = 让用户尝到甜头，然后引导转化**
