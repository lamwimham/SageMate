MCP apps are interactive, AI-native applications designed to run directly inside AI clients rather than as standalone programs or web tabs. They leverage the Model Context Protocol to dynamically access user context, external tools, and live data streams within conversational or agent-driven interfaces.

## Key Characteristics
Unlike traditional software, MCP apps are rendered and operated within AI environments. This enables highly contextual, action-driven experiences where the AI can generate interfaces, execute backend logic, and maintain persistent state across user sessions.
