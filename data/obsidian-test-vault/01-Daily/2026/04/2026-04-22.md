# 2026-04-22 Wednesday

## Morning Journal

Frustrating start. The [[uvicorn --reload]] wasn't picking up file changes. Spent 30min debugging before realizing there were **TWO** server processes on port 8000.

> Lesson: always `lsof -i :8000` before assuming code changes aren't working.

## Bug Hunt: Thinking Events Missing

**Problem**: Backend test confirmed 71 reasoning chunks generated, but SSE output had zero `thinking` events.

**Root cause**: Server wasn't restarted after modifying `generate_text_stream()`. Old process still running.

**Fix**: Kill all processes + restart.

```bash
lsof -i :8000 | grep LISTEN | awk '{print $2}' | xargs kill -9
```
