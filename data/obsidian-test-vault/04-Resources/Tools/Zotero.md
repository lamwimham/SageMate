# Zotero

Reference management tool for academic papers.

## Workflow

1. **Import** → Browser extension captures PDFs
2. **Annotate** → Built-in PDF reader + highlights
3. **Extract** → [[Zotfile]] extracts annotations
4. **Export** → [[Mdnotes]] → Obsidian/SageMate

## Key Plugins

- **Zotfile** — Auto-rename PDFs + extract annotations
- **Better BibTeX** — Stable citation keys
- **Mdnotes** — Export to Markdown

## Integration with SageMate

Goal: automate [[Zotero]] → [[SageMate-Core]] pipeline.

```
Zotero (PDF + annotations)
  → Mdnotes (Markdown)
  → SageMate Compiler
  → Wiki Pages (concepts + source archive)
```
