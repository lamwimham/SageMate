## Workflow

1. **Capture** → Quick Add plugin → 00-Inbox
2. **Process** → Daily review → organize into PARA
3. **Distill** → Literature notes → Permanent notes
4. **Express** → MOCs → Writing projects

## Why Building SageMate

Obsidian is great but lacks:
- Built-in AI Q&A
- Automatic compilation from sources
- Streaming everything

See [[SageMate vs Obsidian]] for full comparison.
