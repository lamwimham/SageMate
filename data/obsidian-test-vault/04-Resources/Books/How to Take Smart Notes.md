# How to Take Smart Notes

**Author**: Sönke Ahrens  
**Published**: 2017  
**Status**: Reading (ch.3/12)  
**Rating**: ⭐⭐⭐⭐⭐

## Core Idea

The [[Zettelkasten]] method as practiced by [[Niklas Luhmann]] — not a filing system, but a **thinking partner**.

## Key Concepts

### Literature Notes

Notes taken while reading. Capture:
- What the author says
- Your own thoughts/reactions
- Context (page, chapter)

> "Write exactly one note for each idea." — Ahrens

### Permanent Notes

Atomic, self-contained notes written in your own words. Each note should:
- Be understandable without context
- Link to other notes
- Be limited to one idea

### Project Notes

Temporary notes related to a specific project. Discarded after project completion.

## Workflow

```
Read → Literature Note → Permanent Note → Connect → Develop → Write
```

## Quotes

> "The slip-box is the medium we think in, not the thing we think about."

> "Every intellectual endeavor starts with a note."
