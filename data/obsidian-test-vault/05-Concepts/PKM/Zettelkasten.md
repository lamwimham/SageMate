# Zettelkasten

A note-taking and knowledge management system developed by [[Niklas Luhmann]].

## Core Principles

### 1. Atomic Notes

Each note contains exactly **one idea**. If you can't explain it simply, split it.

### 2. Unique Identifiers

Luhmann used sequential numbers: `1`, `1a`, `1a1`, `1b`, etc. Modern systems use timestamps or titles.

### 3. Linking

Every new note connects to existing notes. No orphan notes allowed.

### 4. Writing in Your Own Words

Never copy-paste. Reformulate to ensure understanding.

## Types of Notes

| Type | Purpose | Lifespan |
|------|---------|----------|
| Fleeting | Quick capture | Hours-days |
| Literature | Source summaries | Permanent |
| Permanent | Atomic ideas | Permanent |
| Project | Task-specific | Project duration |
