# Second Brain

Term coined by [[Tiago Forte]] for an external digital system that captures and organizes knowledge.

## CODE Framework

**C**apture — Keep what resonates  
**O**rganize — Save for actionability  
**D**istill — Find the essence  
**E**xpress — Show your work

## PARA Method

See [[PARA Method]] for full details.

## Key Insight

> "Your mind is for having ideas, not holding them." — David Allen

The Second Brain frees cognitive RAM by offloading storage.

## vs Zettelkasten

| Aspect | Second Brain | Zettelkasten |
|--------|-------------|--------------|
| Focus | Actionability | Understanding |
| Organization | PARA | Emergent links |
| Note size | Any size | Atomic |
| Output | Projects | Publications |
