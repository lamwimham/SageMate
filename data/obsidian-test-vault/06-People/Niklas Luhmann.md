# Niklas Luhmann

German sociologist, creator of the [[Zettelkasten]] method.

## Contributions

- **Zettelkasten**: 90,000+ index cards → 50+ books, 600+ articles
- **Systems Theory**: Influential social theory
- **Autopoiesis**: Self-organizing systems

## The Slip-Box

Luhmann's index card system had unique features:
- Sequential numbering with branching (1 → 1a → 1a1)
- Index for entry points
- No hierarchy — emergence through linking

## Quote

> "I only do what is easy. I only write when I immediately know how to do it. If I falter for a moment, I put it aside."
